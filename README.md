# "Healthmonitor quality"

В компании "SetCom" требуется создать и поддерживать в актуальном состоянии документацию для нового веб-сервиса "HealthMonitor".
Проект направлен на создание системы мониторинга здоровья пользователей, позволяющей оперативно получать информацию о состоянии организма и давать рекомендации по улучшению качества жизни.
